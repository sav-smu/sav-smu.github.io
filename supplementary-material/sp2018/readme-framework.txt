The framework consists of a simple script. The main function of this software is provided by generate_mutation.py: you can run it by typing "python generate_mutation.py" in a terminal. Before running the code, you need to change the path of the targeted file in the third line.

The SprtTest file is a simple Java program for determining whether or not the model passes statistical model checking. You need to use the function getSampler().sample() to read the data, and the function test() to judge whether or not the sample is a counterexample.
